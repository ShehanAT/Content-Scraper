## Content Scraper

###### This is the sixth of twelve projects for the 🏡 Treehouse Techdegree Full Stack JavaScript.

Crawls and scrapes content from [shirts4mike](http://www.shirts4mike.com/).


On command line: 

`npm install`

then

`npm start`
